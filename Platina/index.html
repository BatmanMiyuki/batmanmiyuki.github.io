Ajouter un compte PStat
Chaque compte garde son propre NPSSO dans GitHub Secrets. Le token n’est jamais saisi ni enregistré dans cette application publique.
1Connectez-vous sur PlayStation.com avec le compte concerné, puis ouvrez le lien NPSSO.
2Dans GitHub Secrets, gardez PSN_NPSSO pour BATMAN_Miyukii et créez PSN_NPSSO_FRERE.
3Lancez l’action « Update PStat accounts » sur « Les deux comptes ».
Obtenir le NPSSO Secrets GitHub
⚠ Ne partagez jamais un NPSSO par message et ne le mettez jamais directement dans le fichier HTML.
